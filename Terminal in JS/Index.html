Terminal Commands

Different Terms

  • Command Line -

    Any Interface that is used by entering textual commands (gen. windows centric)

  • Terminal -

    This is type of Command line (gen. Mac Centric)

  • Console -

    A command-line interface used to work with your computer.

  • Shell -

    A Program running on terminal

  • Bash -

    A Popular shell on mac, OS/Linux

  • Z-Shell -

    Another Shell (default)

Basic Commands

  • ls -

    List Files (Show My Files)

  • pwd -

    Print Working Directory (Where am i?)

  • clear -

    Clear Screen

Navigation Commands -

Inside & Outside Directories

  • cd -

    Change Directory

  • cd Desktop
  • cd .. = Back Button

Paths

Absolute & Relative

  • cd desktop/delta(relative path) - current location
  • cd/user.dine/desktop(absolute path)
  • / = root derectory
  • ~ = home directory

Make Directory - mkdir directory name

Flags

Are characters that we pass with commands to modify their behaviour.

Manual commands

  • man ls -

    give about ls command

  • man mkdir -

    give info about mkdir command

with Flags

  • ls -/
  • ls -a
  • ls -ls
